Flex Heading

Accomplish

Horizontal or Vertical arrangement of elements

Space distribution of elements

Quick Notes

Flex is one-dimensional. It will either affect a row or column but not both at the same time.

Useful for layout on nav bars, headers, image galleries.

Flex places elements that are direct children, but does not affect further children.

Flex Container

Flex containers are elements with the 'display: flex' style. This style applies to the container and its children.

Justifying Content

Justifying content will position content along the axis of our flex container.

  1. flex-start
  2. flex-end
  3. center
  4. space-between
  5. space-around
  6. space-evenly

Align Items

Align items property is similar to justify contents property but it works along the cross-axis (perpendicular to the flex-direction)

  1. flex-start
  2. flex-end
  3. center
  4. stretch
  5. baseline
